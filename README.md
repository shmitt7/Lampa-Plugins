Собственные плагины
(код максимально сжат без всяких проверок/хлама и написан в одну строку)
1. content-labels.js 
Визуальные метки типа контента: синие для фильмов, красные для сериалов | Также показывает где аниме, мульт, реалити, ток-шоу, новости, доку
2. dmca-unblock.js
Разблокировка заблокированного контента в результатах поиска | Долгое нажатие на карточку открывает онлайн-просмотр | Удаляет рекламу в начале фильма для источников где она есть
3. torrent-buttons.js
Управление несколькими парсерами Jackett. Доступ: Карточка → Торренты → кнопка µTorrent в шапке | Долгий тап на карточку → Парсер
4. russian-content.js
Добавляет категорию Русское: новинки, популярное, фильмы, сериалы, мультфильмы, реалити...
5. watch-buttons.js
Раздельные кнопки в карточке фильма | Цветные кнопки торрент и трейлер
